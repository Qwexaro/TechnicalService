## Установка системы

### Требования
- **Node.js** версии 18 или выше (скачать с https://nodejs.org/)
- **Windows 10/11** (для .exe версии)
- 100 MB свободного места на диске

### Способ 1: Установка через Node.js

1. **Скачайте или скопируйте файлы проекта** в любую папку

2. **Установите зависимости** (если нужно):
```bash
npm install
```

3. **Запустите установку**:
```bash
npm start
```
или
```bash
node installer.js
```

4. **Дождитесь завершения установки**. Система установится в папку:
```
C:\Users\[Имя_пользователя]\TechnicalServiceSystem
```

### Способ 2: Использование готового .exe установщика

1. **Запустите файл** `TechnicalServiceSetup.exe`
2. **Дождитесь установки** (файлы будут скопированы в папку пользователя)
3. **На рабочем столе появится ярлык** "ТехОбслуживание"

## Запуск системы

### Способ 1: Через ярлык на рабочем столе
- Дважды кликните по ярлыку **"ТехОбслуживание"**

### Способ 2: Через командную строку
```bash
cd C:\Users\[Имя_пользователя]\TechnicalServiceSystem
start start.bat
```

1. Откройте папку `C:\Users\[Имя_пользователя]\TechnicalServiceSystem`
2. Запустите файл `start.bat`

### Что происходит при запуске:
- Открывается консольное окно
- Запускается веб-сервер на порту `8080`
- Автоматически открывается браузер по адресу `http://localhost:8080`

### Остановка системы:
- Закройте консольное окно или нажмите `Ctrl+C`

## Вход в систему

### Первый вход (для всех пользователей)

При первом входе система **обязательно потребует сменить пароль**!

### Учетные данные по умолчанию:

| Роль | Логин | Пароль |
|------|-------|--------|
| 👨‍💼 **Диспетчер** | `dispatcher` | `dispatcher123` |
| 🏢 **Компании-заказчики** | | |
| | `info@technoprom.ru` | `company123` |
| | `contact@energotech.ru` | `company123` |
| | `info@promreshenie.ru` | `company123` |

### Как войти:

1. Откройте `http://localhost:8080` в браузере
2. Нажмите кнопку **"Вход"**
3. Введите логин и пароль
4. Нажмите **"Войти"**

### При первом входе:
- Появится сообщение: "Необходимо сменить пароль при первом входе"
- Автоматически откроется окно смены пароля
- Введите новый пароль (минимум 4 символа)
- Подтвердите пароль
- После смены пароля откроется основная панель

## Управление инженерами

### Добавление инженера

1. Перейдите на вкладку **"👨‍🔧 Инженеры"**
2. Нажмите кнопку **"➕ Добавить инженера"**
3. Заполните форму:
   - **ФИО** - только буквы (обязательно)
   - **Специализация** (обязательно)
   - **Телефон** (необязательно, формат: +7XXXXXXXXXX)
   - **Email** (необязательно)
   - **Квалификация** (необязательно)
   - **Пароль** (обязательно, минимум 4 символа)
4. Нажмите **"Сохранить"**

### Редактирование инженера

1. На вкладке **"Инженеры"** найдите нужного инженера
2. Нажмите кнопку **"✏️ Редактировать"**
3. Измените данные
4. Нажмите **"Сохранить"**

### Удаление инженера

1. Нажмите кнопку **"🗑️ Удалить"**
2. Подтвердите удаление
3. **Важно:** нельзя удалить инженера, у которого есть активные заявки

### Активация/деактивация инженера

- **🔴 Деактивировать** - инженер не сможет входить в систему и получать заявки
- **🟢 Активировать** - восстановить доступ

## Работа с заявками

### Создание заявки (только для компаний)

1. Войдите как компания-заказчик
2. Нажмите кнопку **"➕ Оформить заявку на ТО"**
3. Заполните форму:
   - **Название заявки** (минимум 3 символа)
   - **Тип оборудования** (минимум 2 символа)
   - **Описание проблемы**
   - **Приоритет** (Низкий/Средний/Высокий/Срочный)
   - **Срок выполнения** (дата не может быть в прошлом)
4. Нажмите **"Отправить заявку"**

### Просмотр заявки

- Нажмите кнопку **"Просмотр"** рядом с любой заявкой
- Откроется окно с детальной информацией

### Одобрение заявки (только для диспетчера)

1. Откройте заявку со статусом "⏳ Ожидает"
2. Нажмите **"✅ Одобрить заявку"**
3. Статус изменится на "✅ Одобрена"

### Отклонение заявки (только для диспетчера)

1. Откройте заявку
2. Нажмите **"❌ Отклонить"**
3. Укажите причину отклонения
4. Нажмите **"Отклонить"**

### Назначение инженера (только для диспетчера)

1. Откройте одобренную заявку
2. Нажмите **"👨‍🔧 Назначить инженера"**
3. Выберите инженера из списка
4. Нажмите **"Назначить"**

### Взятие заявки в работу (только для инженера)

1. В разделе **"🆓 Доступные заявки"** найдите подходящую
2. Нажмите **"🔧 Взять в работу"**
3. Статус изменится на "🔧 В работе"

### Выполнение заявки (только для инженера)

1. Откройте заявку со статусом "🔧 В работе"
2. Нажмите **"✅ Завершить заявку"**
3. Статус изменится на "✔️ Выполнена"
4. Инженеру засчитывается +1 к выполненным заявкам

### Отмена заявки (только для заказчика)

1. Откройте заявку со статусом "⏳ На рассмотрении" или "✅ Одобрена"
2. Нажмите **"❌ Отменить заявку"**
3. Подтвердите отмену

## Отзывы

### Оставить отзыв (только для заказчика)

1. Дождитесь выполнения заявки (статус "✔️ Выполнена")
2. Откройте заявку
3. Нажмите **"⭐ Оставить отзыв"**
4. Выберите оценку (1-5 звезд)
5. Напишите комментарий (минимум 3 символа)
6. Нажмите **"Отправить отзыв"**

## Смена пароля

### Для любого пользователя:

1. Нажмите кнопку **"🔑 Сменить пароль"** (в правом верхнем углу)
2. Введите:
   - **Новый пароль** (минимум 4 символа)
   - **Подтверждение пароля**
3. Нажмите **"Сохранить"**

**Важно:** При первом входе смена пароля обязательна!

## Устранение неполадок

### Ошибка: "Node.js не найден"

**Решение:** Скачайте и установите Node.js с https://nodejs.org/

### Ошибка: "Порт 8080 уже используется"

**Решение:**
1. Закройте другие программы, использующие порт 8080
2. Или измените порт в файле `server.js` (переменная PORT)

### Ошибка: "Не удается войти в систему"

**Решение:**
1. Проверьте правильность логина и пароля
2. Убедитесь, что сервер запущен
3. Попробуйте сбросить базу данных:
   - Откройте консоль браузера (F12)
   - Выполните: `resetDatabase()`

### Ошибка: "База данных не создается"

**Решение:**
1. Очистите кэш браузера
2. Откройте консоль (F12) → Application → IndexedDB
3. Удалите базу `TechnicalServiceDB`
4. Перезагрузите страницу

### Ошибка: "ФИО должно содержать только буквы"

**Решение:** Используйте только русские или английские буквы, пробелы и дефисы

### Ошибка: "Неверный формат телефона"

**Решение:** Используйте формат: `+7(XXX)XXX-XX-XX` или `+7XXXXXXXXXX`

### Ошибка при запуске start.bat

**Решение:**
1. Убедитесь, что файл `server.js` существует
2. Запустите от имени администратора
3. Проверьте, что Node.js установлен: `node --version`

## Поддержка

При возникновении проблем:
1. Проверьте консоль браузера (F12) на наличие ошибок
2. Проверьте консоль сервера (окно start.bat)
3. Выполните сброс базы данных: `resetDatabase()` в консоли браузера
4. Переустановите систему: удалите папку `TechnicalServiceSystem` и запустите установщик заново
